3．用vue-navigation插件时注意的问题
注：vue-navigation是全局性的，只要引入，所有页面都会有缓存问题，如果项目需求是只针对某个页面进行缓存，可以用vue自带的keep-alive组件，具体用法就不在此详细说明了。
当引入vue-navigation时，页面第一次进入，钩子的触发顺序created-> mounted-> activated，退出时触发deactivated。当再次进入（前进或者后退）时，只触发activated。因为 vue-navigation之后页面模板第一次初始化解析成html后，再次进入就不再重新解析而是 读取内存中的数据，就是说只有当数据变化时，才会让页面重新渲染。因此，页面进入时的数据获取应该在activated中也放一份。数据获取完毕手动操作dom的部分也应该在activated中执行才会生效。所以，应该在activated中留一份数据获取的代码，或者不要mounted部分，直接将mounted中的代码全部转移到activated中。同理，当离开当前页面时，需要销毁的一些监听函数，在destroyed里写入的同时应该也在deactivated里写一份，如下：

4．滚动条距离顶部的距离兼容问题
用window.pageYOffset /document.body.scrollTop/ document.docuementElement.scrollTop ,这几种方法只能获取滚动条的距离，不能去赋值，不然有一些低版本的手机系统会报错，导致整个页面都打不开，如果想给滚动条距离赋值，就用window.scrollTop（0,0）

7．懒加载的实现原理
lazyLoad() {
let items = document.getElementsByClassName(‘my-item’)
for (let i in items) {
let item = items[i]
if (typeof item == ‘object’) {
if (this.inViewport(item)) {
let id = item.getAttribute(‘id’)
let viewportItem = this.getItemById(id)
this.onItemShow(viewportItem)
}
}
}
},
getItemById(id) {
return this.allArr.filter(function(item){
return item.id == id
})[0]
},
onItemShow(item) {
item.articleVos.forEach(function(articleItem){
Vue.set(articleItem, ‘showContent’, articleItem.content)
})
},
inViewport(elem, threshold) {
var o = elem.getBoundingClientRect()
var pageWidth = document.documentElement.clientWidth //获取浏览器窗口文档显示区域的宽度，不包括滚动条
var pageHeight = document.documentElement.clientHeight ////获取浏览器窗口文档显示区域的高度，不包括滚动条
threshold = threshold || pageHeight / 5
return o.left < pageWidth + threshold && o.top < pageHeight + threshold
}

 基本原理：侦听window的scroll事件，当页面发生滚动的时候，去遍历所有的item项，判断当前item是否处于窗口之内，如果在窗口之内就去加载，否则不加载，
注意给每个item一个固定的高度，否则所有item就叠在一起，那所有的item都在视窗内，这样就失去了懒加载的意义，另外就是页面一进来就要手动触发一下window的scroll事件，不然一上来不会触发首屏内容的加载

1.getBoundingClientRect()用于获取某个元素相对于视窗的位置集合，包含有top,right,bottom,left等属性，这个方法没有参数，兼容到ie5以上

2.threshold: 是提前加载的高度，自己决定，给多少就代表距离底部多少距离开始加载，以参数的形式传进来是为了提前加载，相当于图片还没进入视窗，就可以提前开始加载了，免得进入视窗之后才开始加载影响用户体验

  1. vue-lazyload 只针对图片进行懒加载 （v-lazy）
  2. 轮播图轮播一圈后，再返回过来是从第二帧开始轮播的，解决方法如下：
    <swiper :options="headOption" ref="mySwiper" v-if="banners.length">

  </swiper>
