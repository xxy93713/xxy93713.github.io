I Wanted To Say

各位领导好：
我叫刘真延，于2017年9月11日成为咱们公司居家生活事业部的一名前端开发人员，目前为止我已来到公司快三个月了，这段时间本人与同事相处融洽，尊敬各个领导。在这近三个月的工作和学习中，接触了不少人和事，学到了很多东西，同时我也明白自己尚有许多缺点需要改正，现在我对我工作的情况做如下汇报：
一、对所负责的岗位的熟悉
主要负责居家生活事业部的前端业务的开发，刚开始对我来说压力有点大，因为我觉得大公司对技术要求比较高，对个人的整体素质要求也高，对于从小公司出来的我来说，很多东西都是全新的接触和认知，这期间特别感谢我们组的每位同事对我的帮助，让我尽快融入集体，尽早认知自己的职责。
二、主要工作内容及成绩
来到公司接手的第一个项目是移动端的预约挂号项目，该项目需要我一个人用 react + webpack 独立完成，说实话，刚开始接手时压力挺大的，因为项目周期比较短，开发时间就一个月，而且又是我一个人完成，心里有点没底。不过随着开发的进展，我发现一切还算顺利，当然中间不免会遇到一些难解的问题和bug，我都耐心的逐条解决，其中有些问题也在同事的帮助下解决掉了，最后完成了项目中所有功能和动态交互，保证了项目的顺利上线。
PC端的艺术品项目是我接手的第二个项目，该项目我主要负责的是商家后台管理页面，艺术品设置页面的电脑版和手机版，运营端上传背景图的电脑版和手机版等所有页面的静态开发和动态交互功能的完成，该项目整体也比较顺利，就是中间产品加了一些需求和改动，拖延了几天，目前也已经接近尾声，下一步要和后端对页面，然后提测上线。
直通名医将是我接手的第三个项目，这个是预约挂号项目的二期，和一期相比，该项目的功能和需求增加了很多，难度也稍微提升了一点，我打算继续用react + webpack 开发，因为挂号的一期我就是用react + webpack开发的，现在二期用同样的框架和技术能提高开发的效率，同时也能让我对react框架更深一步的了解和学习。我目前正在规划整个项目的流程，做好充分的准备，以便后期能够顺利开发完成。
三、下一阶段的计划
近三个月以来在领导和各位同事的指导和帮助下，使我的工作能够顺利进行，在此期间也发现了自己的需要提升几个方面具体如下：
(1) 前端技术瞬息万变，坚持学习各种技术知识，并用于实践。
(2) 对工作的预见性和创造性不够，在今后的工作和学习中，我会进一步严格要求自己，虚心向其他同事学习，争取在各方面取得更大的进步。
时间过得很快，我已到公司工作近三个月。非常感谢公司领导对我的信任，给予我体现自我、提高自我的机会。这近三个月的工作经历，使我的工作能力得到了大幅度的提升。
在对试用期的工作情况及心得体会做一汇报后，我正式向公司领导提出转正申请。希望自己能有幸成为公司的正式一员，做一名真正的京东人。请各领导予以批准，谢谢！
申请人：刘真延
日期：2017年11月20日
